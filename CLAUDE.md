# Card Hero - 프로젝트 가이드

## 프로젝트 개요

- **엔진**: Godot
- **장르**: 카드 + 전투 RPG / 2D 탑뷰 액션
- **핵심 재미**: 다양한 빌드 조합, 덱빌딩, 수집
- **수집/강화 루프**: 조합으로 카드 제작 → 점진적으로 강해지는 적 → 처치 시 소재 획득 → 더 강한 카드 제작 (과금 없이 노가다 기반)

---

## 스토리

카드의 힘으로 살아가는 부족이 있는 탑 내부. 타로 카드들이 탑 곳곳으로 흩뿌려지면서 탑의 시간이 붕괴되고 부족들이 마을을 지킬 힘을 잃는다.

판도라의 상자 카드가 봉인되면서 카드들이 오염됨과 동시에, 4명의 주인공이 카드 오염을 제거하는 능력을 갖게 된다 (정화단). 주인공 일행은 부족들에게 카드를 얻고 흩뿌려진 타로카드를 회수하며 탑의 생태계를 원래대로 복구한다.

- **복원 대상 카드**: 5개 (각 카드 정화에 스테이지/보스 5개씩 필요)
- **총 보스급 몹 목표**: 25~26마리
- **메인 구조**: 큰 맵 하나 + 미니보스 5마리 (난이도 비슷) + 스폰지점 설정

## 캐릭터

| 직업 | 속성 | 성별 | 성격 | 난이도 |
|------|------|------|------|--------|
| 전사 | 🔥 불 | 여 | 불같음 | 중 |
| 탱커 | 🌍 땅 | 여 | 정의로움, 억제기 | 중 |
| 유틸 | ⚡ 전기 | 여 | 쿨미녀 | 상 |
| 법사 | 💧 물 | 남 | 귀여운 거 좋아함, 근육빵빵 | 상 |
| 사제 | 🌿 풀 | 남 | 미남 | 하 |

### 캐릭터별 특징

**전사 (불 🔥)**
- 다양한 무기 교체 전투, 자체 회복 가능
- 코스트/드로우 수급 쉬움
- 모티브: 화투 (불화=꽃화)

**탱커 (땅 🌍)**
- 방패가 몸보다 큰 비주얼
- 국밥탱커 / 회피탱커 두 스타일

**유틸 (전기 ⚡)**
- 무기보다 캐릭터가 커야 함
- 석궁 / 도적 / 양궁 스타일

**법사 (물 💧)**
- 근육질 남성 캐릭터
- 소환술사 / 원소술사 / 정령술사 스타일

**사제 (풀 🌿)**
- 미남 남성 캐릭터
- 버퍼 / 디버퍼 / 성기사 / 힐러 스타일

## 전투 시스템

### 기본공격 / 특수공격 / 회피

| 직업 | 기본공격 | 특수공격 | 회피 |
|------|----------|----------|------|
| 전사 | 베기 | 차지공격 | 대쉬 (짧고 무적시간 적음) |
| 탱커 | 방패치기 | 방패들기 | ❌ |
| 유틸 | ❌ | ❌ | 구르기 (무적 길고, 이동거리 김) |
| 법사 | 완드 휘두르기 | 차징 | 점멸 (쿨타임 있음, 거리 짧음) |
| 사제 | 힐 (쿨타임 김) | 이속버프 (쿨타임 김) | ❌ |

### 코스트 / 드로우 시스템

| 항목 | 내용 |
|------|------|
| 최대 코스트 | 5 |
| 코스트 충전 | 기본공격 or 시간에 따라 충전 |
| 드로우 | 특수공격 or 카드 효과 |
| 손패 | 최대 5장, 넘치면 뽑은 카드 버리기 |
| 카드 분류 | 무기카드 / 패시브 / 액티브 |
| 덱 구성 | 15장, 등급마다 들어갈 수 있는 최대치 다름 |

### 직업별 코스트/드로우 획득 방식
- **전사**: 공격 시 코스트 획득, 특수공격 시 드로우
- **탱커**: 회피 불가 / 완벽한 타이밍 방어 시 1코스트
- **유틸**: 무기 없이 시작 / 회피 시 코스트 획득
- **법사**: 차징 or 근거리 공격 시 (방어력 낮음)
- **사제**: 코스트가 빠르게 참

## 카드 시스템

### 카드 유형
- 무기카드 / 패시브 / 액티브
- 효과 종류: 공격 스킬 / 버프 / 디버프 / 지형지물 / 증강 / 드로우 / 특수효과 / 치유

### 카드 레어도
일반 / 레어 / 유니크 / 전설

### 카드 데이터 구조 (직업별 DB)
- 카드이름, 카드유형 (스킬/무기/파워), 레어도, 코스트, 카드설명, 비고
- 유틸 카드에는 "기본 지급" 여부 필드 있음

### 카드 제작
- 카드 제작 레시피 유사도 기반 (재료 조합)

### 카드 강화 (1회 한정, 던전 안에서 휘발성)
- 선천성 강화 / 코스트 감소 / 데미지 증가 / 지속시간 증가 / 기타 옵션

### 미정 사항
- 한 종류 카드를 몇 장까지 제작 가능하게 할 것인가?
- 강화 시스템 던전 한정 휘발성으로 확정 여부

## 스테이지 디자인

### 구조
- 마을과 분리된 전투공간
- 맵에 웨이포인트, 스테이지마다 평균 난이도 + 구역별 몬스터 풀
- 보스몬스터: 특정 스폰 조건 or 지역에 따라 존재
- 몹 특성에 따라 선공 여부 다름, 추적 범위 설정 (펠월드 스타일)
- 입출입 자유, 퀘스트 중 사망 시 마을 귀환
- 다음 스테이지는 잠겨 있고, 가장 강한 몬스터 처치 등 해금 조건 필요

### 월드 테마 (지역별 속성)
동물 / 기계 / 언데드 / 악마 / RPG세상

### 미정 사항
- 일반 몹 종류 및 속성 정의
- 엘리트 몹 설계
- 보스 몹 설계 (속성별 1종 이상)
- 몹별 드롭 소재 정의
- 던전/스테이지 구성 방식
- 난이도 곡선 설계
- 환경 요소 (지형지물과 카드 연계)

## 프로젝트 구조

```
card-hero/
├── scenes/
│   ├── player/       # 캐릭터별 씬
│   ├── enemies/      # 몬스터 씬
│   ├── ui/card/      # 카드 UI 씬
│   └── dungeon/      # 스테이지/던전 씬
├── scripts/
│   ├── player/       # 캐릭터 스크립트
│   ├── enemies/      # 몬스터 AI
│   ├── cards/        # 카드 로직
│   ├── managers/     # 게임 매니저 (덱, 전투, 스테이지 등)
│   └── components/   # 재사용 컴포넌트
└── assets/
    ├── sprites/      # 스프라이트 이미지
    └── audio/        # 효과음/배경음
```
